## ¿? Patrones de Diseño Implementados

### ¿? Singleton
Implementa el patrón Singleton para garantizar una única instancia de una clase en la aplicación.

¿? Ubicación del código: `Singleton/`

### ¿? Factory Method
Implementa el patrón Factory Method para la creación de objetos sin especificar su clase concreta.

¿? Ubicación del código: `Factory/`
